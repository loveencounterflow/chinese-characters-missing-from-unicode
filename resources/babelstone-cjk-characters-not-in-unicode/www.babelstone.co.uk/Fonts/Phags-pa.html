BabelStone Phags-pa Fonts

Description

BabelStone Phags-pa Book, BabelStone Phags-pa Tibetan A, BabelStone Phags-pa Tibetan B, and BabelStone Phags-pa Seal are different styles of Phags-pa fonts, with OpenType features that allow complex joining and shaping behaviour. There are two versions of the fonts, using different OpenType features. The version 1 fonts (released in 2006) work under Windows XP and Vista with some applications (e.g. Notepad and BabelPad), but do not work at all under Windows 7 or later. The version 2 fonts (released in 2013) work under Windows 7 and later, but do not work correctly under earlier versions of Windows.

Note also that these fonts are designed for vertical layout, and so will be rotated counterclockwise with respect to normal reading orientation when used in horizontal contexts (see Vertical Layout in IE8 for information about how to achieve vertical left-to-right layout in HTML).


BabelStone Phags-pa Book is a general purpose book-style Phags-pa font. See Phags-pa Fonts 1 : BabelStone Phags-pa Book for more information.

BabelStone Phags-pa Tibetan A and BabelStone Phags-pa Tibetan B are two Tibetan-style Phags-pa fonts. See Phags-pa Fonts 2 : BabelStone Phags-pa Tibetan for more information.

BabelStone Phags-pa Seal is a seal script style Phags-pa font. Variant glyph forms of various Phags-pa letters are available in the PUA. See Phags-pa Fonts 3 : BabelStone Phags-pa Seal for more information. This font is not yet available in a version that works under Windows 7 and later.
