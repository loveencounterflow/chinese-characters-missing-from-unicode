Tangut Yinchuan

Description

Tangut Yinchuan v. 11.001 is a font for the Tangut script that supports the full set of Tangut characters defined in Unicode version 11.0 (Tangut code chart). It is based on a font named XXZT (西夏字体 in Chinese) that was designed by Prof. Jing Yongshi 景永时 of the Beifang Ethnic University (北方民族大学) in Yinchuan. The original font was used for typesetting the revised 2nd edition of the Tangut-Chinese dictionary Xià-Hàn Zìdiǎn 夏漢字典 (Beijing, 2008) by Prof. Li Fanwen 李范文.

The original font (XXZT) was remapped to Unicode code points and extended to cover all Unicode Tangut ideographs and components by Andrew West and Michael Everson. We have also made numerous glyph changes, most of which are documented in WG2 N4588 (Tangut glyph corrections by Andrew West, Viacheslav Zaytsev, Sun Bojun, Michael Everson). This modified font is used for Tangut ideographs and Tangut components in the Unicode and ISO/IEC 10646 code charts. Tangut Yinchuan is a fork of the code chart font.

This font has been made available for free download and use with the permission of Prof. Jing Yongshi, who reserves the copyright to the font. This font may not be modified, sold or redistributed for profit without the express permission of the copyright holders. If you use this font in a publication you are requested to acknowledge Prof. Jing Yongshi's copyright of the font.

The font is maintained by Andrew West and Michael Everson, and any feature requests or bug reports should be directed to Andrew West by email or on twitter.

Notes

  • This font is based on the font used for Tangut ideographs and Tangut components in the Unicode and ISO/IEC 10646 code charts. However, glyph corrections have been made to 34 Tangut ideographs and one Tangut component, as documented in Unicode Technical Note #42 (Tangut Character Additions and Glyph Corrections by Andrew West and Viacheslav Zaytsev). Some of these glyph corrections have already been applied to the Unicode 10.0 code charts, and the rest should be applied in Unicode 11.0 and 12.0, but they have all already been applied in this version of the Tangut Yinchuan font. A number of other as yet undocumented glyph changes have also been made (affected characters are listed under Version History for versions 10.000 and 11.000).
  • 11 Tangut characters that have been accepted for encoding in a future version of the Unicode Standard are included in the Private Use Area at U+F000 through U+F00A. The 5 characters at U+F000 through U+F004 are scheduled for inclusion in Unicode version 11.0 at U+187ED through U+187F1 (see WG2 N4922 p. 104), and the 6 characters at U+F005 through U+F00A are provisionally scheduled for inclusion in Unicode version 12.0 at U+187F2 through U+187F7 (see WG2 N4922 p. 66). Infomation about these additional Tangut characters is provided in Unicode Technical Note #42 Three additional candidates for encoding are included at U+F00B through U+F00D.
  • 93 unencoded Tangut components are included in the Private Use Area at U+E000 through U+E216. Of these, the four components at U+E002 through U+E005 are candidates for encoding in the future.
  • 349 variant glyphs for Tangut ideographs are included in the Supplementary Private Use Area at U+F0000 through U+F0160. However, it is anticipated that most of these will be replaced by Ideographic Variation Sequences in a future version of this font. Please note that characters in the Private Use Areas are not stable, and may be moved or removed in future versions of the font.

"Tangut Yinchuan" is the normal font, and "Tangut Yinchuan Inverse" is the same font but with all glyphs rendered as white on a black background (except for U+0020 space, which for weird technical reasons must be transparent; use U+00A0 NBSP instead). Click on the font name to download the TTF font file for installation on your computer or mobile device. The WOFF and WOFF2 files are for web hosting use only, and are not required by most users.
