BabelStone Fonts

BabelStone Shapes

Description

BabelStone Shapes is a font covering various geometric shapes, box drawing shapes, symbols, and arrows defined in the Unicode Standard. The font covers 540 symbol characters, including a number of recent Unicode additions such as power symbols, Go marker symbols, and half star rating symbols. The current version of the font supports Unicode Standard version 11.0.


Characters

The symbol characters in the font are listed below. The unencoded Prince symbol is mapped to U+1F934 "Prince", and the unencoded Symbol of Chaos is mapped to U+2BD1 "Uncertainty sign". Two unencoded vertically oriented half star symbols are mapped to the Private Use Area at U+E059 and U+E05C.
