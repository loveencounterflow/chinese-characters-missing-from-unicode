BabelStone Younger Futhark Fonts

Description

BabelStone Runic is a generic Unicode runic font covering all runic characters in the Unicode Standard version 7.0 (see the Unicode code chart for the Runic block). As this is a generic font, the glyph forms for some characters may not be ideal for some contexts, but is should be a useful fallback font for Runic text. There are two versions of this font:

  • BabelStone Runic : plain runes
  • BabelStone Runic Ruled : the same but with ruled lines above and below each character

Both fonts support a variety of bind runes as Zero Width Joiner (ZWJ) ligatures. Both fonts also include mirrored glyphs for right-to-left layout. Right-to-left layout can be applied to Runic text set with these fonts by putting U+202D (LEFT-TO-RIGHT OVERRIDE [RLO]) at the start of the text and putting U+202C (POP DIRECTIONAL FORMATTING [PDF]) at the end of the text (PDF is not required at the end of a line, as the directionality of the text is reset at a new line, but for the same reason RLO is required at the start of each line of RTL text). This works as expected in Firefox and Chrome browsers, but in IE11 and Edge browsers the text is reordered from right to left, but the mirrored glyphs are not applied.
