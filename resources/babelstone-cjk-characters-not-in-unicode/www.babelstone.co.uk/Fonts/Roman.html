BabelStone Roman

Description

BabelStone Roman is a Unicode serif font covering the Latin script. It is intended primarily for use in phonetic transcription of Tangut, but may be useful for other purposes. The font is derived from Doulos SIL version 5.000, but restricted to the Latin script, with a number of additional characters included. The current version of the font supports Unicode 9.0, and covers 1,299 out of 1,352 Latin script characters in Unicode 9.0.
