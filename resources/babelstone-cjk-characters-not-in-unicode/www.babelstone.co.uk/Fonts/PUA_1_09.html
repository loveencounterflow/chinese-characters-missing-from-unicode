BabelStone Han PUA Version 1.009


Description

This page lists the PUA characters in BabelStone Han PUA version 1.009 that were removed in version 1.010 because they were added to CJK Unified Ideographs Extension E in Unicode version 8.0. The table below gives the mapping from PUA code point in BabelStone Han PUA version 1.009 to the code point in CJK Ext. E. All these characters are available in BabelStone Han version 8.0 and later at the new codepoints. An archive of BabelStone Han PUA version 1.009 is available for download here if required.
